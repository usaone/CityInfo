Open Package Manager Console and execute the following command
PM> Add-Migration CityInfoDBInitialMigration

After migrations created execute the following PM command.

Note: The line "Database.EnsureCreated();" should be commented in CityInfoContext.cs as 
it will conflict by trying to create a DB with tables that are already being updated by this
package manager console command.

PM> Update-Database

First time I experienced some errors shown below and hence deleted the DB manually 
again and re-ran the Update-Database command. Second time it ran successfully.
